# Company Autocomplete

Works with both Vue 2 , Vue 3 And React.

# Translations

* [中文文档](README_zh.md)

## Installing

```bash
# or pnpm or yarn
npm install company-autocomplete
```

## Usage

### General

```ts
import { CompanyAutocomplete } from 'company-autocomplete'

const companyAutocomplete = new CompanyAutocomplete({
  target: '#example'
})
```
